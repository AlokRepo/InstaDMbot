Setup & Troubleshooting Guide

Learn how to retrieve your tokens and configure your Meta app properly.

  1. Go to the Meta Developer Portal and log in.
  2. Click Create App, choose the Business use case (highly recommended for Instagram API integration), and complete app registration.
  3. Make sure your Instagram account is a Business/Creator (Professional) profile.
  4. Verify your Instagram account is linked to your Facebook Page via the Page Settings > Linked Accounts.
  1. Open the Meta Graph API Explorer.
  2. In the Meta App dropdown, select the App you created in Step 1.
  3. In the Permissions section, add these scopes:
    • instagram_business_basic
    • instagram_business_manage_messages (for sending DMs)
    • instagram_manage_comments (for reading/replying to comments)
    • pages_show_list
    • pages_read_engagement
  4. Click Generate Token and authenticate. Copy the resulting token into the "Short-Lived User Access Token" field above.

If you don't want to use an App Secret or deal with page exchanges, you can create a System User in Meta Business Manager:

  1. Go to your Business Settings > Users > System Users.
  2. Add a new System User (admin access recommended) and assign the Facebook Page and Instagram Account assets to them.
  3. Click Generate New Token, select your App, check the permissions listed in Step 2, and generate.
  4. This token **never expires** and can be pasted directly into the Page Access Token field in the manual configuration form below.
